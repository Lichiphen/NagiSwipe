# NagiSwipe

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

軽量画像ポップアップモーダルです。

## 特徴

- **ドロップイン導入**: JSとCSSを読み込むだけで、ページ内の画像リンクを自動的にギャラリー化します。
- **モバイル最適化**: スワイプ、ピンチズーム、ダブルタップに対応。
- **軽量・高速**: 依存ライブラリなし。
- **スムーズなアニメーション**: 60fpsを目指した滑らかな操作感。

## 使い方

HTMLに以下の2ファイルを読み込むだけです。

```html
<link rel="stylesheet" href="NagiSwipe-main.css">
<script src="NagiSwipe-main.js"></script>
```

ページ内の `<a href="image.jpg">` のような形式のリンクが自動的に検出され、クリック時にギャラリーが開きます。
